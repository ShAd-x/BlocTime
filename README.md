### BlocTime est une application mobile simple pour suivre son temps quotidien de manière visuelle.

Chaque journée est découpée en blocs de 30 minutes, auxquels tu peux assigner une catégorie (Travail, Sommeil, Loisirs, etc.).

Le but est de prendre conscience de son emploi du temps, identifier les habitudes et mieux gérer son temps.


#### ⚠️ Ce projet est personnel et en cours de développement. Il s’agit d’un MVP pour tester le concept, apprendre SwiftUI et expérimenter une interface de type “time blocking”.
